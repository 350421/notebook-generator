## 2026-07-06

### 优化

- 优化撤销交互：
  - 文本输入、误删文本、模板固定文字编辑改为优先走细粒度撤销
  - `Ctrl + Z / Cmd + Z` 会尽量只撤回上一步输入或删除，不再默认整段回退
  - 撤销按钮会优先撤回当前文本编辑，再回退图片、素材、块顺序等结构性操作
- 新增重做交互：
  - 支持页面“重做”按钮
  - 支持 `Ctrl + Y / Ctrl + Shift + Z / Cmd + Shift + Z`
  - 文本编辑优先细粒度重做，结构性操作继续使用快照前进
- 补齐文字局部格式操作的撤销历史，避免加粗、改色、改字号后无法正确回退
- 优化分页与标题编号：
  - 小标题编号改为跨页连续递增，不会每页重新从 `01` 开始
  - 分页时优先保证“小标题 + 下一段内容”同页，减少页尾孤立小标题
- 优化封面体验：
  - 开启封面模式后，即使不勾选“封面独立第一页”，首页也会自动整理为封面效果
  - “优先使用封面图”会优先抓取显式封面图；如果没有显式封面图，则回退使用第一张可用图片
  - 封面主标题和副标题样式重新调整，减少标题过大、重复感和排版生硬的问题
- 优化预览交互：
  - 微调区改字、加粗、改色、改字号、封面设置、模板样式调整后会自动触发预览同步
  - 不再必须手动点击“重新生成预览”才能看到最新页面

### 优化（分页稳定性 - 2026-07-06 第二轮）

- 长列表可跨页拆分：
  - 超过 3 条的列表块在分页前自动按 2-3 条一组拆分为子块，使分页器可自然跨页
  - 有序列表拆分后通过 HTML start 属性延续编号，不会从 01 重新开始
  - 拆分只影响渲染内部，不改 blocks 数据模型，不影响前端
- 最后一页稀疏检测：
  - 当最后一页内容不足可用高度 30% 时，尝试从倒数第二页末尾回移 1 个 block
  - 减少最后一大片留白突兀感
- 分页失败错误提示产品化：
  - 现在会指明具体哪个内容块过长、该块高度多少、单页可用高度多少
  - 用户能根据提示直接定位并拆分对应段落
